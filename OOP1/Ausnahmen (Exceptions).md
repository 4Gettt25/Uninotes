## Eigenschaften
- Ausnahmen werden verwendet, um unerwartete Situationen während der Ausführung zu behandeln.
- Beispiel für Try-Catch:
	```java
	try {
    int result = 10 / 0;
	} catch (ArithmeticException e) {
    System.out.println("Division durch Null nicht erlaubt.");
	}
 ```
